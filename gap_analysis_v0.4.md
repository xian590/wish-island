## 二、缺失的关键素材（按优先级排序）

### 🔴 P0：缺少就无法做核心玩法的素材

#### 1. 迷你游戏专用素材（4套，约16张）
> **这是目前最大的缺口**。参考炼金模拟器，迷你游戏是核心操作乐趣来源，但我们的听诊器/研磨游戏全靠代码画几何图形，没有美术素材支撑。

| 迷你游戏 | 缺失素材 | 数量 | 为什么必须 |
|---------|---------|------|-----------|
| **听诊器** | 宠物身体轮廓（卡通化）、心跳波形背景、听诊器光标 | 3张 | 游戏主视觉，没有=简陋 |
| **研磨草药** | 石制研磨钵（俯视）、草药碎屑粒子、研磨进度环 | 3张 | 操作反馈的核心视觉 |
| **缝合手术** | 伤口俯视图、缝合线虚线、判定区标记、手术刀光标 | 4张 | 手术室场景的核心玩法 |
| **配药混合** | 透明烧杯、红/蓝/黄药液（可叠加）、目标色卡圆盘 | 4张 | 药房核心玩法 |
| **温度计** | 温度计本体（卡通）、温度刻度、红色液柱 | 2张 | 检查室玩法 |

**P0合计：约16张**

## 四、与三大参考游戏的核心系统差距

### 4.1 杯杯倒满差距

| 杯杯倒满核心 | 我们当前 | 差距 |
|-------------|---------|------|
| **多客人同时排队** | 一次只接待1个 | ❌ 缺少并发压力 |
| **进货→备料→制作→出货** | 只有对话→点击治疗 | ❌ 缺少经营流程深度 |
| **肉鸽：每天不同buff** | 有事件但无buff图标 | ⚠️ 有框架缺表现 |
| **多人合作** | 单机 | ⚠️ 暂不考虑 |
| **丰富的UI动效** | 基础淡入淡出 | ❌ 缺少粒子/特效 |
| **服装/皮肤系统** | 无 | ❌ 长期目标 |

### 4.2 东方夜雀食堂差距

| 夜雀食堂核心 | 我们当前 | 差距 |
|-------------|---------|------|
| **白天收集+夜晚经营双阶段** | 只有营业阶段 | ❌ 缺少早晨准备阶段 |
| **顾客羁绊Lv.1-5** | 有框架但无深度 | ⚠️ 缺少专属剧情/解锁 |
| **口味匹配** | 无差异化治疗 | ❌ 缺少策略深度 |
| **符卡：特殊客人特殊事件** | 有随机事件但简单 | ⚠️ 缺少惊喜感 |
| **昼夜氛围变化** | 有夜景但无切换逻辑 | ❌ 缺少时间流逝感 |
| **弹幕式评价** | 结算面板 | ❌ 缺少即时反馈爽感 |
| **支线剧情** | 无 | ❌ 缺少叙事深度 |
| **季节变化** | 无 | ❌ 缺少长期变化 |

### 4.3 炼金模拟器差距

| 炼金模拟器核心 | 我们当前 | 差距 |
|---------------|---------|------|
| **物理交互式制作** | 2个迷你游戏 | ⚠️ 只有2个，需4-6个 |
| **配方探索地图** | 固定处方 | ❌ 缺少探索感 |
| **原料组合实验** | 无组合系统 | ❌ 缺少实验乐趣 |
| **讨价还价** | 固定诊金 | ❌ 缺少经济策略 |
| **药瓶自定义外观** | 无 | ❌ 低优先级 |
| **药剂效果可视化** | 无 | ❌ 缺少直观反馈 |
| **丰富的原料种类** | 5种道具 | ❌ 原料太少 |
| **销售策略** | 无 | ❌ 缺少深度 |

## 六、核心判断

**现在最缺的是：**
1. 🔴 **迷你游戏专用素材**（16张）——没有这些，听诊器/研磨游戏永远像半成品
2. 🔴 **特效素材**（8张）——没有这些，治愈/收钱/升级缺少爽感反馈

**这两个补齐后，游戏就能达到"可玩数小时"的门槛。**

NPC第2批、天气、成就这些可以并行出，不影响核心循环。
