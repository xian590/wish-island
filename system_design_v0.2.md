# 宠物医生模拟器 — 差距分析与系统方案 v0.2

## 一、当前原型 vs 参考游戏：诚实差距

### 1. 杯杯倒满（Feed The Cups）
| 杯杯倒满的核心 | 我们目前的原型 | 差距 |
|---|---|---|
| **多客人同时排队**，手速管理 | 一次只接待1个宠物，无时间压力 | ❌ 缺少并发压力 |
| **进货→备料→制作→出货完整流程** | 只有"对话→点击治疗" | ❌ 缺少经营流程深度 |
| **压力系统**：错误太多会爆棚 | 无压力机制 | ❌ 缺少紧张感 |
| **轻度肉鸽**：每天不同buff/debuff | 固定流程 | ❌ 缺少重玩价值 |
| **多人合作**（可选） | 单机 | ⚠️ 暂不考虑 |

### 2. 东方夜雀食堂
| 夜雀食堂的核心 | 我们目前的原型 | 差距 |
|---|---|---|
| **白天收集+夜晚经营双阶段** | 只有诊所场景，无时间循环 | ❌ 缺少节奏变化 |
| **顾客羁绊等级**：解锁新菜谱/对话 | 无羁绊系统 | ❌ 缺少长期目标 |
| **口味匹配**：不同客人有不同喜好 | 无差异化治疗 | ❌ 缺少策略深度 |
| **符卡机制**：特殊客人有特殊事件 | 无随机事件 | ❌ 缺少惊喜感 |
| **昼夜氛围变化** | 无时间系统 | ❌ 缺少沉浸感 |

### 3. 炼金模拟器（Potion Craft）
| 炼金模拟器的核心 | 我们目前的原型 | 差距 |
|---|---|---|
| **物理交互式制作**：研磨/搅拌/加热/加水 | 点击按钮直接治疗 | ❌ 缺少操作乐趣 |
| **配方探索地图**：自己发现组合 | 固定处方 | ❌ 缺少探索感 |
| **原料组合实验**：不同比例不同效果 | 无组合系统 | ❌ 缺少实验乐趣 |
| **讨价还价**：与顾客谈价格 | 固定诊金 | ❌ 缺少经济策略 |
| **药瓶自定义外观** | 无 | ⚠️ 低优先级 |

## 二、v0.2 完整系统方案

### 核心循环：双阶段制（东方夜雀食堂式）

```
┌─────────────────────────────────────────────────────────────┐
│  🌅 阶段一：早晨准备（7:00-9:00）                              │
│  ── 查看今日预约（随机生成3-5个病例）                           │
│  ── 采购药品/检查库存（金币管理）                               │
│  ── 花园采集草药（种植系统，可升级）                            │
│  ── 与NPC对话（解锁剧情、获取线索）                            │
│  ── 升级诊所设备（声望解锁）                                   │
├─────────────────────────────────────────────────────────────┤
│  🏥 阶段二：诊所营业（9:00-18:00）                             │
│  ── 宠物主人带宠物上门                                        │
│  ── 对话问诊（选择问题→获取线索）                              │
│  ── 诊断迷你游戏（听诊/测温/触诊/X光）                         │
│  ── 治疗迷你游戏（配药/研磨/注射/包扎）                        │
│  ── 结算评价（治好→声望+金币；治坏→声望下降）                  │
│  ── 客人排队压力（同时2-3个客人在等）                          │
├─────────────────────────────────────────────────────────────┤
│  🌙 阶段三：夜晚总结（18:00-20:00）                            │
│  ── 查看今日收支                                               │
│  ── 处理住院宠物（如果有病房系统）                              │
│  ── 随机事件触发（流行病/特价/稀有客人预告）                    │
│  ── 存档/结束一天                                              │
└─────────────────────────────────────────────────────────────┘
```

### 2.1 压力系统（杯杯倒满式）

**诊所压力条**（0-100%）：
- 客人等待时间过长 → 压力+5/分钟
- 诊断错误 → 压力+15
- 宠物病情恶化 → 压力+20
- 连续治好3个 → 压力-10（成就感缓冲）

**压力后果**：
- 0-30%：正常运营
- 30-60%：客人开始不耐烦（对话框变红，有催促选项）
- 60-80%：医生操作失误率上升（迷你游戏判定区缩小）
- 80-100%：客人愤然离开，声望大跌，当天强制结束

### 2.2 诊断迷你游戏（炼金模拟器式物理交互）

**迷你游戏1：听诊器找心跳**
- 鼠标控制听诊器在宠物身上移动
- 屏幕上显示声波图，越靠近心脏波形越规律
- 找到正确位置后按住3秒完成
- 难度：生病宠物心跳不齐，波形混乱

**迷你游戏2：温度计**
- 点击插入→等待上升→在正确区间点击读取
- 过快取出：数值不准
- 过慢：宠物不耐烦，压力+5

**迷你游戏3：研磨草药**
- 鼠标按住画圈，圈速决定研磨程度
- 进度条显示：太粗/正好/太细
- 不同病症需要不同粗细

**迷你游戏4：混合药剂**
- 拖放不同颜色的药液进烧杯
- 颜色混合后需要匹配目标色（RGB混合逻辑）
- 配错颜色：药效下降或产生副作用

**迷你游戏5：包扎**
- 点击开始→绷带自动滚动
- 在绿色判定区点击停止，完美包扎
- 过早/过晚：绷带太松或太紧

### 2.3 病例与声誉系统

**病例档案**：
```
┌────────────────────────────────┐
│  病历号 #042                    │
│  患者：豆豆（柴犬·3岁·雄）      │
│  主人：林先生（羁绊Lv.3）        │
│  初诊日期：Day 12               │
│  病史：食物中毒（Day 8复发一次） │
│  过敏：对青霉素过敏 ⚠️          │
│  治疗方案：黄连素+益生菌        │
│  预后：良好                     │
└────────────────────────────────┘
```

**声誉等级**：
- ⭐ 1星：默默无闻（日接诊1-2个）
- ⭐⭐ 2星：小有名气（日接诊3-4个，解锁药房升级）
- ⭐⭐⭐ 3星：口碑诊所（日接诊4-5个，解锁手术室）
- ⭐⭐⭐⭐ 4星：全市知名（日接诊5-6个，解锁稀有宠物）
- ⭐⭐⭐⭐⭐ 5星：传奇兽医（日接诊6-8个，解锁隐藏剧情）

### 2.4 顾客羁绊系统（东方夜雀食堂式）

| 羁绊等级 | 效果 |
|---------|------|
| Lv.1 陌生人 | 基础对话，无特殊 |
| Lv.2 熟客 | 解锁额外病情信息，更容易诊断 |
| Lv.3 朋友 | 送来稀有草药，介绍新客人 |
| Lv.4 挚友 | 宠物生病优先通知你，解锁专属剧情 |
| Lv.5 家人 | 免费帮你宣传，解锁隐藏结局线索 |

**提升羁绊方式**：治好宠物、多聊天、记住宠物名字、送小礼物

### 2.5 肉鸽元素：每日随机事件

| 事件类型 | 效果 |
|---------|------|
| 🦠 流感爆发 | 今天所有宠物+30%得呼吸道疾病的概率 |
| 💰 药品特价 | 今天采购药品半价 |
| 🌟 明星宠物 | 某网红博主的宠物来看病，治好+大量声望 |
| ⚡ 停电 | 检查室设备不可用，只能用基础诊断 |
| 🎁 神秘包裹 | 收到匿名捐赠的稀有草药 |
| 😷 医生生病 | 你今天操作失误率+20%（除非休息） |

### 2.6 经济系统

**收入来源**：
- 基础诊金：¥50-200/次（按难度）
- 药品费用：成本价+利润
- 手术费用：¥500-2000（解锁手术室后）
- 住院费：¥30/天（解锁病房后）

**支出项目**：
- 药品采购
- 设备维护
- 花园种子
- 诊所租金（每周扣）
- 升级费用

**破产线**：金币<0且持续3天 → 游戏结束（Bad Ending）

## 三、呼吸动效设计

所有角色（医生、宠物、NPC）都需要呼吸动效：

```css
/* 呼吸动效：身体轻微缩放+上下浮动 */
@keyframes breathe {
    0%, 100% { 
        transform: translateY(0) scale(1, 1); 
    }
    25% { 
        transform: translateY(-3px) scale(1.01, 0.99); 
    }
    50% { 
        transform: translateY(-4px) scale(1.015, 0.985); 
    }
    75% { 
        transform: translateY(-2px) scale(1.005, 0.995); 
    }
}
/* 周期3秒，模拟正常呼吸频率 */
```

- 吸气：身体变宽变矮（胸部扩张）
- 呼气：身体变窄变高
- 同时伴随2-4px的上下浮动
- 不同角色呼吸频率不同（狗快、人正常）

- ✅ 双阶段循环（简化版：只有营业阶段）
