## 关键 Bug 发现与修复

### 🚨 P0: app.js IIFE 阻断所有交互功能

**问题**: 拆分后的 `app.js` 被包裹在 `(function() { ... })()` IIFE 中，导致所有函数（`showPage`, `goBack`, `StorageUtil`, `initApp` 等）无法暴露到全局作用域。HTML 中的 `onclick="showPage('vision')"` 等内联事件处理器全部失效，页面显示 "初始化遇到一点小问题，请刷新页面重试"。

**根因**: 从单文件提取 JS 时，保留了原始代码中不存在的 IIFE 包装。

**修复**: 移除 `app.js` 首尾的 IIFE 包装：
- 第 4 行：删除 `(function() {`
- 第 12404 行：删除 `})();`

**验证**: `node --check` 语法通过，浏览器刷新后 `hasShowPage: true`, `hasStorageUtil: true`。
