# 宠物医生模拟器 v0.3 — 浏览器冒烟测试指南

## 前置准备

1. **启动本地 HTTP 服务器**（在项目根目录）：
   ```bash
   cd C:/Users/Administrator/Documents/kimi/workspace
   python -m http.server 8080
   ```
   保持此窗口运行。

2. **在浏览器中打开**：
   ```
   http://localhost:8080/pet_doctor_v0.3.html
   ```
   推荐浏览器：Chrome / Edge（最新版）。

## 深度测试建议（可选，约 30 分钟）

### 压力爆表测试
1. 故意连续失败治疗，让压力升到 100
2. 观察：所有等待病人离开，当天强制结束，声望大幅扣除

### 破产测试
1. 在调试面板中将金币设为负数
2. 连续 3 天保持负数
3. 观察：第 3 天触发破产结局

### 季节切换测试
1. 在调试面板中快速跳转天数（Day 1→30→60→90）
2. 观察：花园采集量变化，场景色调变化（如有 CSS）

### 100天结局测试
1. 在调试面板中跳到 Day 100
2. 观察：结局动画/文字，评级是否正确

### 长时间挂机测试
1. 让游戏在浏览器中运行 1 小时+
2. 观察：内存是否泄漏（Chrome 任务管理器），CPU 占用是否稳定

## 问题反馈流程

发现任何异常时：

1. **按 D 键打开调试面板**
2. **点击「复制诊断报告」**
3. **将报告粘贴给我**，同时描述：
   - 你做了什么操作
   - 期望发生什么
   - 实际发生了什么
   - 是否可复现
