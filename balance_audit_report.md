## 1. 执行摘要

共发现 **14项平衡性问题**，其中：
- **严重 (S)** 2项：技能Lv10不可达、新手期现金流危机
- **高 (H)** 5项：作物ROI两极分化、钓鱼/挖矿收益过低、离线上限过低、房屋价格跳跃、自动化回报不足

- **低 (L)** 2项：集市商品价格倒挂、部分里程碑效果微弱

## 4. 技能升级经验需求（严重问题）

### 问题 4-1: 技能Lv10经验需求过高，几乎不可达

| 位置 | 当前数值 | 问题说明 |
|------|----------|----------|
| `SKILL_EXP_TABLE` (line 6368) | `[0, 25, 85, 200, 400, 700, 1150, 1750, 2650, 4000, 6000]` | Lv10需累计6000经验 |
| `studySkill()` (line ~18252-18302) | 每次读书: 1-3经验, 随机分配给5个技能之一 | 单个技能日均约1.2经验 |
| | 每天最多读书2次 | 有林老师书+1，村小学+2 |

**计算:**
- 假设最优情况：每天读2次，平均每次3经验，全部分配到目标技能（实际不可能，是5选1随机）
- 最优情况需：6000 / 6 = 1000天 ≈ 2.7年（现实时间）
- 实际情况（5技能随机分配）：约5000天 ≈ 13.7年
- 即使只做技能5级的成就（需700经验），也需要约117-350天

**建议调整:**
1. **大幅降低经验表**：`[0, 10, 30, 70, 150, 300, 500, 800, 1200, 1700, 2300]` (line 6368)
2. **增加经验获取途径**：
   - 每次种地操作+2农具精通经验
   - 每次浇水+1田间管理经验
   - 每次施肥+2堆肥经验
   - 每次钓鱼+5钓鱼经验（当前已是+5）
3. **读书技能选择优化**：允许玩家选择要读的技能书，而非完全随机

**原因:** 游戏设计目标不应需要玩家玩十几年才能达到满级。技能应在30-60天内可达满级，以配合游戏后期的100-365天目标。

## 7. 离线收益

### 问题 7-1: 离线上限仅1天，放置收益严重不足

| 位置 | 当前数值 | 问题说明 |
|------|----------|----------|
| `calculateOfflineReward()` (line ~9893-10808) | `MAX_OFFLINE_DAYS = 1` (line 9933) | 最多推进24游戏小时 |
| | 离线自动出售所有库存作物 (line ~10751-10770) | 按当前季节价格出售，可能低价卖出 |
| | 每天扣30元饭钱，没钱给15元救济 (line ~10013-10020) | 低保机制防止破产，但玩家离线上限后收益极低 |

**建议调整:**
1. 离线上限提升至3天（72小时）或7天（168小时）(line 9933)
2. 增加离线保护期：前30分钟离线不计入，避免频繁刷新
3. 离线自动出售增加价格保护：只在价格>=基础价90%时出售，否则保留库存 (line ~10756-10760)
4. 或改为离线只自动收获不自动出售，让玩家上线后决定何时卖

**原因:** 1天上限对放置型玩家极其不友好。如果玩家睡8小时离线，游戏只推进约16小时（游戏速度下），可能连一个作物周期都完成不了。放置类游戏应允许至少3-7天的离线积累。

## 10. 游戏后期（30天+）目标与玩法

### 问题 10-1: 后期缺乏足够的消费目标和玩法深度

| 位置 | 当前数值 | 问题说明 |
|------|----------|----------|
| 最高价建筑 | 庄园: 200000元 | 但买完后只剩农家乐、科技、集市等次要系统 |
| `AGRI_TECH_DATA` (line ~5328-5341) | 最高科技: 太空育种 200000元 | 一次性购买，之后无持续目标 |
| `VILLAGE_PROJECTS` | 村长项目最高15000元 | 声望限制，30天后可能已买完 |
| 成就 | 最终目标 survive365 | 但中间200-300天缺乏阶段性目标 |

**后期玩家状态（30天后）:**
- 房屋可能已升至精英房或终极房
- 有1-2块额外田地
- 养了若干牲畜
- 所有NPC已认识
- 科技树可能已点到中期
- **问题：每天就是重复种地->收获->卖钱，缺乏新系统解锁**

**建议调整（新增后期目标）:**
1. **新增农场评级系统**: 30天后解锁，从县级示范农场->省级农场->国家级农场，每级需满足产量/收入/设施/声望要求
2. **新增农产品展销会**: 每季度举办，需要提交特定高品质作物，获奖可得大量声望和金钱
3. **新增后代培养系统**: 100天后解锁，培养孩子接手农场，引入新一代玩法
4. **增加科技树的无限层数**: 当前10层，50天后可解锁星际农业等趣味后期科技（高成本低回报，但有趣味性）
5. **新增天气灾害挑战模式**: 30天后可选，高回报但高风险
