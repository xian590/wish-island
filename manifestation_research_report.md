## 四、v6.0 功能设计规格

### 4.1 369 显化法模块

**设计理念**: 基于TikTok爆火的369方法，将数字的神秘感与书写仪式感结合。

**核心机制**:
- 用户输入一条肯定语（affirmation）
- 早晨写3次 → 午后续写6次 → 晚上续写9次
- 连续21天或33天为一个周期
- 每天完成时获得水晶奖励
- 支持周期进度跟踪

**UI设计**:
- 主页面: 今日进度条(3/6/9)、历史 streak、当前周期天数
- 书写页面: 简化输入框，自动计数，进度显示
- 周期页面: 日历网格，显示每天完成情况

**数据结构**:
```javascript
{
  affirmation: "string",
  cycleStart: "date",
  cycleLength: 21/33/45,
  dailyRecords: { "2026-06-01": { morning:3, noon:6, evening:9 } },
  currentStreak: 0,
  longestStreak: 0
}
```

### 4.2 55x5 显化法模块

**设计理念**: 5天集中冲刺，高强度重复印刻潜意识。

**核心机制**:
- 用户输入一条肯定语
- 每天写55次，连续5天
- 每次书写有计数器和进度条
- 第5天完成时显示庆祝动画
- 支持多次挑战（可重复进行）

**UI设计**:
- 主页面: 当前挑战天数(1-5)、今日进度(0/55)、历史挑战记录
- 书写页面: 大型输入框，每次书写自动计数，第55次有特殊反馈
- 完成页面: 庆祝动画、成就徽章

**数据结构**:
```javascript
{
  affirmation: "string",
  currentDay: 1-5,
  dailyCount: 0-55,
  completedChallenges: [{ startDate, endDate, affirmation }],
  totalChallenges: 0
}
```

### 4.3 显化证据记录模块

**设计理念**: 类似Lumina的Daily Sign和Manifestive的Signs，帮助用户建立"显化正在发生"的信心。

**核心机制**:
- 记录日常生活中观察到的同步性、巧合、信号
- 分类标签: 金钱/爱情/健康/机会/梦境/直觉/其他
- 支持文字记录+日期+情绪强度(1-5星)
- 统计图表: 每月证据数量、分类分布
- 支持搜索和筛选

**UI设计**:
- 主页面: 最新证据列表、添加按钮、统计卡片
- 添加页面: 分类选择、文字输入、情绪强度、日期
- 统计页面: 图表（时间线+分类饼图）
- 灵感页面: 提示用户"今天注意什么信号"

**数据结构**:
```javascript
{
  signs: [{ id, date, category, text, intensity, createdAt }],
  categories: ["money", "love", "health", "opportunity", "dream", "intuition", "other"]
}
```

### 4.4 Focus Wheel 聚焦轮模块

**设计理念**: Abraham-Hicks经典工具，从负面核心信念出发，通过层层递进的正面陈述，逐步提升情感振动。

**核心机制**:
- 用户输入一个核心负面信念/担忧
- 系统生成12个位置的聚焦轮
- 中心: 负面核心信念
- 外圈12个位置: 逐步提升的正面陈述
- 每个位置由用户填写或AI辅助生成
- 完成整个轮后，情感振动应该显著提升

**UI设计**:
- 主页面: 创建新轮、历史轮列表
- 编辑页面: 圆形交互界面，中心负面信念，12个位置可点击编辑
- 完成页面: 情感前后对比、保存和分享

**数据结构**:
```javascript
{
  wheels: [{
    id, date, centerBelief,
    statements: ["s1", "s2", ..., "s12"], // 12个递进陈述
    emotionBefore: 1-10, emotionAfter: 1-10
  }]
}
```
