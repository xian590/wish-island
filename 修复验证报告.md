## 一、已修复问题清单

### 1. `showModal` 内联样式兼容性修复 ✅

**问题**: `showModal(id)` 函数使用 `el.style.display = 'block'` 设置内联样式，与 CSS 类控制的 `.show { display: block !important; }` 冲突，导致部分浏览器中模态框不显示。

**修复** (index-manifestation.html:5243):
```javascript
// 修复前
function showModal(id) { var el = document.getElementById(id); if (el) { el.style.display = 'block'; el.classList.add('show'); } }

// 修复后
function showModal(id) { var el = document.getElementById(id); if (el) { el.style.display = ''; el.classList.add('show'); } }
```

**验证**: 
- 设置面板（齿轮图标）→ 模态框正常弹出
- 塔罗次数用完提示 → 模态框正常弹出
- 所有设置选项（声音、主题、白噪音等）正常显示

### 8. `startAudioScene()` 定时器泄漏修复 ✅

**问题**: `startAudioScene()` 函数中 `audioGuideInterval` 在重复调用时（如快速切换场景）会被覆盖赋值，导致旧的 `setInterval` 无法被清理，造成定时器泄漏。

**修复**: 在函数开头增加防护逻辑：
```javascript
function startAudioScene(idx) {
  if(audioGuideInterval) { clearInterval(audioGuideInterval); audioGuideInterval = null; }
  // ... rest of function
}
```

**验证**: 定时器变量分析确认 `audioGuideInterval` 现在有完整的 set/clear 配对。
