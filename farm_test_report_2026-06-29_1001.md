## 三、Bug 发现与修复

### 3.1 Bug 描述：任务数存档与 UI 显示不一致

| 项目 | 修复前 | 修复后 | 状态 |
|------|--------|--------|------|
| 存档任务键数 | 21 | **51** | ✅ 已修复 |
| UI 显示待完成 | 46 | **51** | ✅ 已修复 |
| 一致性 | 不一致（21 ≠ 46） | **一致（51 = 51）** | ✅ 已修复 |

**Bug 严重程度**: 🔴 高（数据不一致，影响玩家信任度）  
**Bug 持续天数**: 从 2026-06-28 日志首次记录，持续至少 **2 天** 未修复。

### 3.2 根因分析

经代码审计，发现 **3 个根本原因** 导致此不一致：

1. **`initGame` 硬编码初始化不全**（行 5873）
   - 新存档只初始化了 `q0` 到 `q20`（21个键），而 `QUEST_DATA` 定义了 `q0` 到 `q50`（51个键）。
   - 缺失的 `q21` 到 `q50` 在后续游戏过程中从未被补全到存档中。

2. **`loadGame` 补全代码硬编码且遗漏**（原行 6872）
   - 补全数组使用硬编码的 `questKeys`，未基于 `QUEST_DATA` 动态生成。
   - 如果 `QUEST_DATA` 新增或删除任务，补全代码将不同步。

3. **`renderQuestBox` 渲染数组硬编码且存在缺陷**（行 9588）
   - **ID 冲突**：`devQuests` 错误将"乔迁之喜"标记为 `q7`，与 `newbieQuests` 的 `q7`"开市售卖"冲突，导致 `allQuests` 中 `q7` 被重复计算，而 `q8`（真正的乔迁之喜）被遗漏。
   - **数组缺失**：`q13` 到 `q17`（成长任务：扩土开疆、田间能手、村民挚友、小有积蓄、德高望重）**完全未加入** `allQuests` 展开，导致 UI 中永远看不到这5个任务。
   - 综合计算：`allQuests` 总长度 = 46（含重复 `q7`），`QUEST_DATA` 键数 = 51，差异 = 5个遗漏 + 1个重复 = 6个任务未正确渲染。

### 3.3 修复方案

#### 修复 1：动态初始化新存档任务（`initGame`）

```javascript
// 修改前（行 5873）：硬编码 21 个任务
quests: { q0: false, ..., q20: false }

// 修改后：基于 QUEST_DATA 动态生成全部 51 个任务
quests: (() => {
    const all = {};
    Object.keys(QUEST_DATA || {}).forEach(id => { all[id] = false; });
    return all;
})()
```

#### 修复 2：动态补全加载的旧存档（`loadGame` 后）

```javascript
// 修改前（行 6872）：硬编码 45 个 questKeys
const questKeys = ['q6', 'q7', ..., 'q50'];
questKeys.forEach(key => { ... });

// 修改后：基于 QUEST_DATA 动态补全
if (game.quests) {
    Object.keys(QUEST_DATA || {}).forEach(key => {
        if (game.quests[key] === undefined) game.quests[key] = false;
    });
}
```

#### 修复 3：修正 `renderQuestBox` 中的 ID 冲突与数组缺失

```javascript
// 修正 devQuests 中 q7 → q8（乔迁之喜实为 q8）
{ id: 'q8', name: '乔迁之喜', ... }

// 新增 growthQuests 数组（q13-q17）
const growthQuests = [
    { id: 'q13', name: '扩土开疆', ... },
    { id: 'q14', name: '田间能手', ... },
    { id: 'q15', name: '村民挚友', ... },
    { id: 'q16', name: '小有积蓄', ... },
    { id: 'q17', name: '德高望重', ... },
];

// 将 growthQuests 加入 allQuests 展开
const allQuests = [..., ...buildQuests, ...growthQuests, ...socialQuests, ...];
```
