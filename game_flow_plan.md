## 二、阶段一：开荒期（Day 1-5）

**目标**：跑通第一块田的完整早稻循环，获取第一桶金

### Day 1 - 整地+育苗

1. 选择【耕读传家】模式
2. 确认状态：资金1500，体力100/200，仓库2000斤
3. 点击【田地管理】→ 第1块田【整地】（-15体力）
4. 点击【商店】→ 默认种子页签 → 买早稻种子1亩（-50元）
5. 点击【田地管理】→ 第1块田【育苗】→ 选早稻（-10体力）
6. 点击【日常活动】→ 【锻炼身体】（-5体力，健康+2~4）
7. 加速到夜间，等待Day 2
8. 检查：资金1430，体力约80，健康+2，早稻育苗中

### Day 2 - 移栽

1. 体力恢复到约80
2. 点击【田地管理】→ 【移栽】（-18体力）
3. 点击【串门聊天】→ 选任意NPC（-10体力，好感+1~3）
4. 点击【河边钓鱼】（-15体力，随机得鱼）
5. 加速到夜间
6. 检查：资金1410，体力约47，早稻进入transplanting

### Day 3 - 浇水+除草

1. 体力恢复到约67
2. 点击【浇水】（-8体力）
3. 点击【除草】（-15体力）
4. 点击【锻炼身体】（-5体力）
5. 加速到夜间
6. 检查：资金1390，体力约49，早稻进入growing

### Day 4 - 浇水+等待

1. 体力恢复到约69
2. 点击【浇水】（-8体力）
3. 点击【看书学习】（-15体力，技能+1~3）
4. 加速到夜间
5. 检查：资金1370，体力约56，早稻接近成熟

### Day 5 - 丰收日 ⭐

1. 体力恢复到约76
2. 点击【田地管理】→ 【收割】（-20体力）
3. **检查任务弹窗**："第一次收割"→ 点击【领取奖励】+200元
4. 检查仓库：水稻约500斤
5. 点击【商店】→ 出售作物 → 全部出售
6. 检查资金：约1370 + 200 + 750 = 2320元
7. 点击【日常活动】→ 【锻炼身体】→ 检查概率事件

**阶段一总结**：5天后资金约2320元，体力约56，掌握完整种田循环

## 三、阶段二：扩地与副业启动（Day 6-15）

**目标**：租第2块田，建造第一个副业建筑

### Day 6 - 租地日

1. 资金约2320，第2块田租金200元
2. 点击【田地管理】→ 底部【租用新田】（-200元）
3. **检查仓库增长**：2000→3000斤
4. 给第2块田【整地】（-15体力）
5. 买早稻种子（-50元）
6. 第2块田【育苗】（-10体力）
7. 第1块田【整地】→ 【育苗】（第2季早稻）

### Day 7-10 - 双田循环

- 每日操作：浇水、除草、聊天、钓鱼
- 每日检查：资金、体力、仓库、作物状态
- Day 10：第2块田成熟，收割约500斤

### Day 11 - 建造副业 ⭐

1. 资金约：2320 - 200(租地) - 50(种子) - 20×10(餐费) + 750(第2季) = 约2770
2. 点击【农家副业】→ 【建造】→ 选择【石磨碾米】（-150元）
3. 检查：每日自动加工稻谷→糙米（需体力≥8）
4. 继续种植第3季早稻

### Day 15 - 春薯考虑

1. 资金约3000+
2. 决定：继续早稻（周转快）或尝试春薯（产量高）
3. 建议：新手期继续早稻，熟练后尝试春薯

## 五、阶段四：深加工（Day 31-60）

## 七、关键决策点

| 决策 | 时机 | 建议 |
|------|------|------|
| 扩地 | 资金≥租金×1.5 + 200预留 | 每5-8天扩一块 |
| 开启副业 | 有原料库存 + 资金充裕 | 早稻收获后先开碾米 |
| 出售时机 | 仓库>40%或资金<200 | 早上6点自动检查 |
| 体力管理 | <15时停止种田 | 优先收割，再整地 |
| 任务领取 | 弹窗出现时立即领取 | 避免错过奖励 |
| 冬季策略 | 停止种田，休息+锻炼 | 等待春天 |

## 八、严格测试流程

### 测试原则
1. 发现问题 → 立即修复
2. 修复后 → 重开测试档（新游戏）
3. 确认修复 → 继续推进
4. 全程确认无问题 → 5倍速循环测试（数小时）

### 测试检查清单（每次操作后）

**UI检查**：
- [ ] 状态栏资金、体力、健康、仓库显示正确
- [ ] 田地状态显示正确（idle/prepared/seedling/transplanting/growing/mature）
- [ ] 商店页签顺序：种子→道具→出售→农副产品→礼物
- [ ] 副业页面显示所有加工品和建筑
- [ ] 村民页面显示所有10个NPC

**数值检查**：
- [ ] 每次操作后资金变化正确（考虑每日餐费）
- [ ] 体力消耗和恢复正确
- [ ] 仓库容量限制生效（超出部分就地处理）
- [ ] 扩地后仓库容量+1000斤
- [ ] 加工品产出正确（考虑概率事件）
- [ ] 出售价格正确（季节性波动）

**事件检查**：
- [ ] 事件文本中的NPC名字正确（孙妙青/林晓雨等）
- [ ] 事件效果正确（好感度增加正确NPC）
- [ ] 任务完成弹窗正确显示
- [ ] 领取奖励后资金正确增加
- [ ] 概率事件文本正确显示

**系统检查**：
- [ ] 自动种田模式下所有操作正常
- [ ] 锁定解锁后不再重新锁定
- [ ] 水稻移栽2天后进入growing（bug已修复）
- [ ] 仓库超过40%自动出售触发
- [ ] 冬季跳过种田逻辑正确

### 测试执行步骤

1. **新游戏开局** → 选择普通模式
2. **Day 1-5手动操作** → 每步检查上述清单
3. **Day 6-15半自动** → 手动关键操作，日常加速
4. **Day 16+ 5倍速挂机** → 每10天检查一次状态
5. **发现问题** → 立即停止，记录问题，修复，重开
6. **确认无问题** → 持续5倍速循环，观察数小时
