## 测试前准备

### 前置检查清单

| # | 检查项 | 操作 | 通过标准 |
|---|--------|------|----------|
| P1 | 清除旧存档 | `localStorage.removeItem('farm_game_save_v1')` + 刷新页面 | 页面回到故事开头 |
| P2 | HTTP 服务器运行 | `curl http://localhost:8082/farm_game.html` | 返回 200 |
| P3 | WebBridge 连接 | `curl http://127.0.0.1:10086/command` | 返回 daemon 响应 |
| P4 | 控制台无报错 | 打开 DevTools Console | 无红色 error |

### 操作步骤
1. 清除 localStorage 中的旧存档
2. 刷新页面，等待故事界面出现
3. 点击"跳过故事"按钮
4. 选择 **Normal 模式（耕读传家）**
5. 验证初始状态（见 Day 1 检查）
